Git-команды

  • git init - инициализирует новый Git-репозиторий в текущей директории.
  • git add - добавляет изменения из рабочей директории в индекс (staging area). Подготавливает файлы к следующему коммиту.
  • git commit -m "сообщение" - создает коммит с добавленными изменениями. Сообщение должно кратко описывать, что изменено.
  • git status - показывает текущее состояние рабочей директории.
  • git push - отправляет локальные коммиты в удаленный репозиторий. Синхронизирует ваши изменения с другими участниками проекта.
